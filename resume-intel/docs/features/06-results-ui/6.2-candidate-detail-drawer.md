## 1. Problem Statement

The results table summarizes each candidate; recruiters need a slide-in detail view with every extracted field, search findings, public records, AI summary, and source quotes — without leaving the main workflow.

## 2. Goals

- Slide-in drawer from “View Full Detail” row action
- Sections: extracted resume fields, search findings, public records, AI summary, outreach suggestions
- `all_employers` rendered as a visual timeline
- Collapsible raw source quotes per field (anti-hallucination)

## 3. Functional Requirements

- **FR-1.** Drawer MUST open/close via button, backdrop click, and Escape.
- **FR-2.** Employment timeline MUST list every role with company, title, date range.
- **FR-3.** When search data exists, drawer MUST show DDG snippets, LinkedIn profile data, and public API hits in separate labeled sections.
- **FR-4.** AI summary MUST show provider label (Gemini / Claude).

## 4. Acceptance Criteria

- [ ] Every parsed JSON field visible in drawer
- [ ] Timeline matches `all_employers` order from parser
- [ ] Search and summary sections appear after Milestones 2–4 ship (graceful empty states until then)
